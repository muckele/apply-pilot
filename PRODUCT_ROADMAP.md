# Product Roadmap

Recommended future features:

- Chrome extension or bookmarklet for manual job saving
- Calendar integration for interview reminders
- Salary comparison
- Company research brief
- Recruiter contact tracker
- Networking CRM
- LinkedIn profile optimization checklist
- Weekly job-search plan
- Application quality score
- Follow-up cadence automation
- AI mock interview mode
- Resume A/B testing
- Kanban pipeline board
- Data export to CSV
- Backup and delete-my-data controls
- Mobile-friendly PWA
- Saved search alerts
- Duplicate company detection
- Referral tracker
- Offer comparison tool

Suggested milestones:

1. Harden manual import, resume parsing, and AI matching.
2. Add editable resume/cover letter document editor with polished exports.
3. Complete Gmail save-to-CRM workflow and reviewed draft handoff.
4. Add interview prep workflows, consented transcription, and feedback loops.
5. Add calendar reminders, saved search alerts, and data export/delete controls.
